Aplicar Regresión lineal al conjunto de datos.¶

Uso la libreria Scikitlearn de modelo linear com su importacion de Regresssion Linear para ajustar el modelo, eso y mis datos extraidos de X e Y solo debo de calcular el coeficiente de regresion en base a un entrenamiento de mi modelo en base a los datos en X e Y. Despues de tener mi modelo entrenado y los datos graficare una linea con matplot de color rojo en base a la predicccion de mi modelo en X.

In [25]:
from sklearn.linear_model import LinearRegression

# Asegrate de tener solo las columnas 'X' e 'Y' en tu conjunto de datos
X = dataset[['x']]
y = dataset['y']

# Inicializar y ajustar el modelo de regresin lineal
model = LinearRegression()
model.fit(X, y)

# Imprimir los coeficientes de la regresin
print("Coeficiente:", model.coef_)
print("Intercepto:", model.intercept_)

# Graficar los datos originales en puntos
plt.scatter(X, y, color='green', label='Data')
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Regression Linear')

# Graficar la lnea de regresin
plt.plot(X, model.predict(X), color='red', linewidth=2, label='Regression Linear')

# Mostrar la leyenda
plt.legend()

# Mostrar el grfico
plt.show()

Aplicar Regression cuadratica al conjunto de datos.¶

Nuevamente con Scikit learn voy a usar el PolynomialFeatures que me permiticra una regresion cuadratica en base a los grados que yo ingrese, en este caso como es una Regresion cuadratica, usare degree=2. Con ello primero creare un polinomio a entrenar en base a los datos en X. Luego de eso con una regresion linear entrenare un modelo en base al polinomio creado junto con el conjunto de datos en Y.

In [27]:
from sklearn.preprocessing import PolynomialFeatures

# Asegrate de tener solo las columnas 'X' e 'Y' en tu conjunto de datos
X = dataset[['x']]
y = dataset['y']

# Ajustar un modelo de regresin cuadrtica 2 grados
poly = PolynomialFeatures(degree=2)
X_poly = poly.fit_transform(X)

model = LinearRegression()
model.fit(X_poly, y)

# Imprimir los coeficientes de la regresin
print("Coeficiente:", model.coef_)
print("Intercepto:", model.intercept_)

# Graficar los datos originales en puntos
plt.scatter(X, y, color='green', label='Data')
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Regression cuadratica')

# Graficar la lnea de regresin
plt.plot(X, model.predict(X_poly), color='red', linewidth=2, label='Regression cuadratica')

# Mostrar la leyenda
plt.legend()

# Mostrar el grfico
plt.show()

from sklearn.preprocessing import PolynomialFeatures

# Asegrate de tener solo las columnas 'X' e 'Y' en tu conjunto de datos
